📌 プロジェクト概要

本プロジェクトは、複数のExcelファイルに分散したデータを読み込み、必要な情報を抽出・集約する業務自動化ツールです。
UiPathを用いて、外部設定ファイルをもとに処理対象や出力先を柔軟に切り替えられる構成としています。
また、処理結果に応じたフォルダ作成やファイル出力、エラー発生時のメール通知など、一連の業務フローを自動化しています。

※本リポジトリは実務での自動化処理をベースに作成していますが、機密情報保護のため、データ・ファイルパス・一部ロジックをサンプル化および簡略化しています。

⚙️ 使用技術
UiPath（Studio）
Excel（データ入出力）
Outlook（メール通知）
DataTable / Dictionary（データ管理）

🧩 主な機能
複数Excelファイルの読み込み・データ集約
外部設定ファイルによる処理制御（柔軟な設計）
フォルダ自動生成
条件分岐による処理制御
エラー時のメール通知
ファイル存在チェックによる重複防止
 
🔄 処理フロー
外部設定ファイルの読み込み
対象Excelファイルの取得
データ抽出・整形
集約データの生成
フォルダ作成およびファイル出力
条件に応じた分岐処理
エラー発生時のメール通知

💡 工夫した点
外部設定（Dictionary）を利用した柔軟な設計
→ ファイルパスや処理対象をコード変更なしで切り替え可能
ワークフロー分割による可読性・保守性向上
→ Main / サブ処理を分離
再利用性を意識した構造
→ 他業務にも流用可能
エラー時の通知設計
→ 業務運用を意識した設計
🚀 想定ユースケース
Excelベースの業務データ集約
定型業務の自動化
手作業による集計業務の削減
